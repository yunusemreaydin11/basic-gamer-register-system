# Projenin İşleyişi
1. Kullanıcıdan kayı için bilgi alır.
2. Bilgileri uygun değişkenlere atar.
3. Liste yöntemi kullanarak bunları düzenler.
4. Dize yöntemi olan .format ile ekrana yazdırır. Bu şekilde dinamik bir dize oluşturur.
